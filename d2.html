2. How to convert int to String.

SOURCE CODE:-

 
        class Method3
{ 
  public static void main(String args[]) 
  { 
    int number = -1234; 
    String str = String.format("%d", number);
    System.out.println("With format method: string = " + str);
  } 
}

OUTPUT:-


  With format method: string = -1234
