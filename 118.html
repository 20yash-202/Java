118. Write a Java program to print Fibonacci series of n terms where n is input by user using loop.

class fibonaci2{
void fibonacci(int a, int b, int c){
System.out.print(a+","+b+",");
for(int i=2; i< c; i++){
int d= a+b;
System.out.print(d+",");
a=b; b=d;
}
}}
