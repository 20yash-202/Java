107. Write a java program to calculate the sum of first 10 natural number using loop.

SOURCE CODE:-

class sumnatural{
public static void main(String s[]){
int i=1;
int sum=0;
while(i< 11){
sum= sum+i;
i++;
}
System.out.println("Sum of first 10 natural numbers is: "+sum);
}}
