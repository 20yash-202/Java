71. Write a Java program to convert all characters in a string to uppercase.

SOURCE CODE:-

class upper{
public static void main(String st[]) throws java.io.IOException{
java.io.InputStreamReader isr= new java.io.InputStreamReader(System.in);
java.io.BufferedReader br= new java.io.BufferedReader(isr);
System.out.println("Enter String1: ");
String str1= br.readLine();
System.out.println("String after converting to upper case is: "+ str1.toUpperCase());
}}
