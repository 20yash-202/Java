JAVA PROGRAMS
1. Write a Java method to find the smallest number among three numbers.

2.Write a Java method to compute the average of three numbers.

3. Write a Java method to find all twin prime numbers less than 100.

4. Write a Java method to display the current date and time.

5. Write a Java method to check whether a string is a valid password.

6. Write a Java method to print characters between two characters (i.e. A to P).

7. Write a Java method to compute the future investment value at a given interest rate for a specified number of years.

8. Write a Java method to count all the words in a string.

9. Write a Java method to display the middle character of a string. Note: a) If the length of the string is odd there will be two middle characters. b) If the length of the string is even there will be one middle character.
