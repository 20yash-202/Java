7. Create a program, showing an example of super function

SOURCE CODE:-

 
        class Animal {

            // overridden method
            public void display(){
              System.out.println("I am an animal");
            }
          }
          
          class Dog extends Animal {
          
            // overriding method
            @Override
            public void display(){
              System.out.println("I am a dog");
            }
          
            public void printMessage(){
          
              // this calls overriding method
              display();
          
              // this calls overridden method
              super.display();
            }
          }
          
          class Main {
            public static void main(String[] args) {
              Dog dog1 = new Dog();
              dog1.printMessage();
            }
          }
          
OUTPUT:-

          
            I am a dog
            I am an animal
